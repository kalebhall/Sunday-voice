# Sunday Voice – Roadmap

## MVP

- Local admin/operator accounts and RBAC.
- Session management (ad hoc + scheduled).
- Whisper API transcription.
- Browser Web Speech fallback.
- Multi-language translation with provider abstraction.
- Anonymous listener access via QR / short URL / code.
- Presentation/closed-caption mode.
- Scrollback during session.
- 48-hour retention enforcement.
- Basic metrics and audit logging.

## Post-MVP

- Self-hosted Whisper.
- Better TTS (multiple providers, caching).
- More languages and improved quality tuning.
- Remote/online meeting integrations (Zoom, etc.).
- Multi-tenant support (multiple stakes).
- Advanced presentation customization.
- Device compatibility tuning after field testing.
